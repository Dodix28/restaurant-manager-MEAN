\\# DineFlow



\\\*Online orders, reservations, and real-time table management for restaurants and cafes.\\\*







---







\\# Description



DineFlow is a full-stack application for restaurants and cafes that allows customers to place online orders and make reservations while giving employees a live overview of table availability. The app features a visual restaurant layout, showing free and occupied tables on a canvas-based interface. It includes an admin panel for managing restaurants, employees, and customers. This application improves restaurant operations, streamlines order management, and enhances the overall customer experience.





---







\\# Technologies



| Frontend | Angular, TypeScript, CSS/HTML |



| Backend | NodeJS, Express |



| Database | MongoDB |



| Other | JSON import/export, REST API |







---







\\# Features

Online Orders – Customers can place and track orders in real time.



Table Management – Visual layout shows free and occupied tables for staff.



Reservations – Clients can book tables easily online.



Admin Panel – Manage restaurants, employees, and customers efficiently.



Responsive Design – Works on desktop, tablet, and mobile devices.



Search \& Filter – Quickly find orders, tables, or clients.
